    About This Project

    Study Hub is a practice website built as part of INFO 1300: Introduction to Web Design and Programming at Cornell University.

    The goal of this site is to create a useful resource hub for college students while practicing key web design and development concepts.

    Built With

    • HTML - semantic structure
    • CSS - responsive design, flexbox, media queries
    • JavaScript - client-side interactivity
    • jQuery - simplified DOM manipulation

    Design Process

    1. Identify Audience

    Primary audience: College students looking to organize and share study resources.

    2. Understand Goals

    • Find study materials quickly by subject
    • Access study tools and templates
    • Develop better study habits
    • Collaborate with other students (future feature)

    3. Content & Organization

    Used card sorting to organize content into pages: Home, Subjects, Resources, Tools, and About.

    4. Design & Sketch

    Designed responsive layout for narrow (mobile) and wide (desktop) screens. Used familiar layout patterns: header navigation, feature cards, footer.

    5. Code

    Implemented with semantic HTML, responsive CSS (flexbox, media queries), and client-side JavaScript interactivity.

    6. Test & Evaluate
